# Numerbook Calculator

> 一个现代化的文本式实时计算器

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

## ✨ 特点

- **实时计算** - 输入表达式，立即看到结果
- **变量支持** - 定义变量并在后续表达式中引用
- **简洁界面** - 精致的现代设计，专注计算体验
- **零依赖** - 纯原生 JavaScript，无需任何构建工具
- **键盘友好** - 完整的键盘快捷键支持

## 📖 使用指南

### 基本计算

直接输入表达式，每行右侧会实时显示结果：

```
10 + 20         → 30
100 / 4         → 25
(1 + 2) * 3     → 9
-5 + 10         → 5
```

### 变量赋值

支持变量定义和引用：

```
a = 10          → 10
b = 20          → 20
c = a + b       → 30
d = c * 2       → 60
```

### 中文变量名

支持中文变量名：

```
单价 = 99
数量 = 5
总计 = 单价 * 数量    → 495
```

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `Enter` | 新增一行 |
| `Backspace` | 删除空行（光标在行首时） |
| `↑` | 聚焦上一行 |
| `↓` | 聚焦下一行 |

## 🧮 支持的运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 | `10 + 20` |
| `-` | 减法 | `20 - 10` |
| `*` | 乘法 | `6 * 7` |
| `/` | 除法 | `100 / 4` |
| `()` | 括号优先级 | `(1 + 2) * 3` |
| `-` (一元) | 负号 | `-5` |

## ❌ 错误处理

| 错误 | 说明 | 示例 |
|------|------|------|
| 变量 'x' 未定义 | 使用了未定义的变量 | `x + 1` |
| 除数不能为零 | 除以零 | `10 / 0` |

错误会以红色高亮显示，不影响其他行的计算。

## 📝 示例场景

### 购物计算

```
单价 = 99
数量 = 5
折扣 = 0.9
运费 = 10
总计 = 单价 * 数量 * 折扣 + 运费    → 455.5
```

### BMI 计算

```
身高 = 1.75
体重 = 70
BMI = 体重 / (身高 * 身高)    → 22.86
```

### 单位转换

```
美元 = 100
汇率 = 7.2
人民币 = 美元 * 汇率    → 720
```

## 🛠️ 技术实现

- **解析器**：手写递归下降解析器，正确的运算符优先级处理
- **变量管理**：基于 Map 的变量存储，支持依赖追踪
- **增量计算**：变量变化时只重新计算相关的行
- **UI 框架**：零框架，纯原生 HTML/CSS/JS
