# Numerbook Calculator - 产品方案与实施计划

## 一、产品定义

**名称**：Numerbook Calculator

**核心体验**：用户在文本框中输入计算表达式，每行右侧实时显示结果

**目标用户**：普通用户（无需编程知识）

**使用场景**：日常计算、账单核对、简单公式计算等

## 四、解析器设计

### 1. 语法定义（BNF）

```bnf
program     ::= line*
line        ::= assignment | expression

assignment  ::= identifier "=" expression
expression  ::= term (("+" | "-") term)*
term        ::= factor (("*" | "/") factor)*
factor      ::= number | identifier | "(" expression ")" | unary
unary       ::= "-" factor | factor
identifier  ::= [a-zA-Z_][a-zA-Z0-9_]*
number      ::= [0-9]+ ("." [0-9]+)?
```

### 2. 支持的运算符

| 运算符 | 优先级 | 示例 |
|--------|--------|------|
| `()` | 1 | `(a + b) * 2` |
| 一元 `-` | 2 | `-5`, `-a` |
| `*` `/` | 3 | `a * b`, `10 / 2` |
| `+` `-` | 4 | `a + b`, `c - 1` |

### 2. 依赖追踪

```
a = 10      → 无依赖
b = 20      → 无依赖
c = a + b   → 依赖 [a, b]
d = c * 2   → 依赖 [c] → 间接依赖 [a, b]
```

**作用**：当 `a` 变化时，只重算 `c` 和 `d`

## 六、错误处理

### 1. 错误类型

| 错误 | 触发条件 | 提示文案 |
|------|----------|----------|
| 未定义变量 | `c = x + 1` (x 未定义) | `变量 'x' 未定义` |
| 除零错误 | `a = 10 / 0` | `除数不能为零` |
| 语法错误 | `a = 1 +` | `意外的 token` |

### 2. 错误显示策略

- 行内显示错误，红色背景高亮
- 错误行不影响其他行的计算
- 用户修正后自动重新计算
